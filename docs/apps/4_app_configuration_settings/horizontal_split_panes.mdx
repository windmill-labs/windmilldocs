# Horizontal split panes

Split panes component is a container split in x number of panes horizontally.

	className="border-2 rounded-lg object-cover w-full h-full dark:border-gray-800"

<br />

To add a component to a horizontal split pane, you can either click on `Insert` while you selected the horizontal split pane, or you can move an existing component by copy/pasting it.

The following section details Horizontal split panes component's specific settings. For more details on the App editor, check the [dedicated documentation](../0_app_editor/index.mdx) or the App editor [Quickstart](../../getting_started/7_apps_quickstart/index.mdx):
